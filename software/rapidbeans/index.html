RapidBeans is an

  • object oriented
  • model driven
  • lightweight

rapid application development framework created by Martin Blümel

RapidBeans' main goal is to let you build an application rapidly out of a domain model consisting in:

  • classes (entities, beans) including
    • attributes (properties)
    • associations
    • constraints
  • and some supporting value types:
    • quantities to combine a magnitude and a unit
    • enumerations which serve to define choices and units

RapidBeans is based on Java and consists of:

  • a core layer defining the type system
  • a simple XML based persistence layer
  • a generic Multiple Document Interface like presentation layer (current implementation: Swing)

... or to speak more in pictures:
You only define a domain model like that
with a view lines of XML and get an application like that
